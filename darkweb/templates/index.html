About Dark Web Recon

Dark Web Recon is an easy to use tool that uses a web interface to configure search modules to scour the dark web for various keywords. It then sends the results to a parser to sift through the data. From there, it sends the information in realtime to the ELK stack for further visualization and analyzation.
